## Quick Start

For new developers joining the project:

1. **Environment Setup**: Start with [Docker Development](/internal-docs/docker-development)
2. **Project Overview**: Read [Project Architecture](/internal-docs/project-architecture)
3. **Development Patterns**: Review [AdonisJS Patterns](/internal-docs/adonisjs-patterns)
4. **Frontend Development**: Study [React & Inertia Patterns](/internal-docs/react-inertia-patterns)

## Contributing

When adding new features or apps to the ecosystem:

- Follow the established [Multi-App Ecosystem](/internal-docs/multi-app-ecosystem) patterns
- Use the shared [UI Components](/internal-docs/ui-components) library
- Leverage [Shared Entities](/internal-docs/shared-entities) from the main app
- Integrate with the [Database & AI](/internal-docs/database-ai-integration) infrastructure

## Support

For questions about these patterns or architecture decisions, reach out to the core development team or create an issue in the project repository.
